Prefer New Line Before Return

ID: prefer_newline_before_return

Severity: style

View all Rules

  • View allRules

Description

Enforces blank line between statements and return in a block.

Example

Bad:

  if ( ... ) {
    ...
    return ...;
  }

Good:

  if ( ... ) {
    return ...;
  }

  if ( ... ) {
    ...

    return ...;
  }
